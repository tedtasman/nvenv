# nvenv

`nvenv` is a simple Python virtual environment manager.  
It simplifies creating, updating, and activating virtual environments. It is designed to work in **bash** and **zsh**

---

## Installation

### Install directly using curl:

_Always review remote installation scripts before executing them._

```bash
curl -fsSL https://raw.githubusercontent.com/tedtasman/nvenv/main/install.sh | bash
```

### Local install (via git clone)

**1. Clone Repository**

```bash
git clone https://github.com/tedtasman/nvenv.git
cd nvenv
```

**2. Run installer**

```bash
./install.sh
```

Follow prompts to optionally add to PATH and make callable without `source`

---

## Removal

An uninstaller is provided. From the repository, run:

```bash
./uninstall.sh
```
